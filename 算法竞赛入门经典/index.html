2-5 分数化小数

本题的难点在于如何精确到小数点后c位。

以2/5为例。

2/5的整数位为0；2%5 = 2;余数为2，2*10=20；20/5整数位为4；20%5 = 0；……

那么，

m = a/b；//保存整数

n = a%b; //保存余数

n = n *10; //余数进位

m = n / b;

n = n % b;
